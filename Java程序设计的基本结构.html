switch\tt switchswitch表达式

与int a=5; 一样,表达式可以将一个值赋给一个变量
  • 在switch 表达式中,可以使用yeild 关键字,与break 类似,yeild 会终止执行,但与break 不同的是,其还会返回一个值,作为表达式的值
  • 不能再switch 表达式中使用return,break,continue 这三个关键字,如:default→{return -1;} 这样写是错误的
import java.io.*;
import java.lang.*;
import java.util.*;
@SuppressWarnings("fallthrough")
public class MyfirstExample {
    public static void main(String[] args)
    {
        Scanner in= new Scanner(System.in);
        String test=in.next();
        String result= switch (test)
        {
            case "yes":
                System.out.println("yes!");
                yield "yes!";
            case "No":
                System.out.println("no!");
                yield "no!";
            default:
                System.out.println("hohohooh!");
                yield "default!";
        };
        System.out.println(result);
    }
}

         System.out.println(BigInteger.ZERO);//0
     System.out.println(BigInteger.ONE);//1
     System.out.println(BigInteger.TWO);//2
     System.out.println(BigInteger.TEN);//10
遗憾的是,Big_Integer 类中没有重载+,−,×,÷+,-,\times,\div+,−,×,÷运算符,java 也不能重载任何的运算符
  • 在需要用到加减乘除的运算操作时,需要使用add,subtract,multiply,divide 等方法,如下:
        BigInteger a=new BigInteger("394723743274893274");
        BigInteger b=new BigInteger("374732743274987329847327847324732432");
        System.out.println(a.multiply(b));
  • 在使用实数运算时,可能会遇到除不尽的操作,为此需要定义舍入方式,常见的舍入方法是四舍五入
    需要注意的是,舍入方法只能用于BigDecimal 而不能用于BigInteger 
            BigDecimal a=BigDecimal.valueOf(10,4);
        BigDecimal c=BigDecimal.valueOf(3,4);
        System.out.println(a.divide(c, RoundingMode.HALF_UP));//四舍五入
    其中构造BigDecimal 时候需要两个参数,一个是原始数字,另一个是其参与运算时候保留的精度

声明方法

在C++\tt C++C++中我们喜欢用int a[] 来定义一个数组变量,在java\tt javajava中,我们更倾向于使用int[] a 来完成这一操作
  • 可以使用int[] a=new int 100; 来定义一个长度为100100100的数组,也可以使用变量创建数组,和C++\tt C++C++是一致的

访问数组元素

  • 和C++\tt C++C++一样,下标从000开始,可以使用.lenth 方法来替代数组元素个数

for each\tt for\ eachfor each循环

  • 可以使用for each 循环遍历数组中的每个元素,而不是通过索引值
        String[]a={"apple","banana","grape","cow","milk","abandon"};
        for(String tmp:a)
        {
            System.out.printf(tmp+" ");
        }
        //apple banana grape cow milk abandon
  • 可以使用Arrays.toString() 方法来讲一个数组转换成一个字符串,如下:
import java.io.*;
import java.lang.*;
import java.util.*;
import java.math.*;
@SuppressWarnings("fallthrough")
public class MyfirstExample {
    public static void main(String[] args)
    {
        String[]a={"apple","banana","grape","cow","milk","abandon"};
        System.out.println(Arrays.toString(a));
    }
}//[apple, banana, grape, cow, milk, abandon]
请注意是使用Arrays.toString(a) 而不是a.toString() 否则会打印出a 的地址

数组的拷贝

  • 如果使用等于运算符进行拷贝,则两个数组变量将对应同一个数组,如下:
        String[]a={"apple","banana","grape","cow","milk","abandon"};
        String[]b=a;
        b[2]="substituted!";
        System.out.println(Arrays.toString(a));//[apple, banana, substituted!, cow, milk, abandon]
  • 如果只希望是值拷贝,可以使用Arrays.copyOf 方法
        String[]a={"apple","banana","grape","cow","milk","abandon"};
        String[]b=Arrays.copyOf(a,a.length);
        b[2]="substituted!";
        System.out.println(Arrays.toString(a));
        System.out.println(Arrays.toString(b));
        //[apple, banana, grape, cow, milk, abandon]
        //[apple, banana, substituted!, cow, milk, abandon]
  • 也可以使用Arrays.copyOf 方法增加原数组的长度
        String[]a={"apple","banana","grape","cow","milk","abandon"};
        System.out.println(a.length);//6
        a=Arrays.copyOf(a,2*a.length);
        System.out.println(a.length);//12

数组排序

  • 使用Arrays.sort() 方法对数组进行升序排序,也可以通过下面的方式进行自定义排序
        String[]a={"apple","banana","grape","cow","milk","abandon"};
        Arrays.sort(a,(c,d)->c.length()-d.length());//按长度排序
        System.out.println(Arrays.toString(a));
